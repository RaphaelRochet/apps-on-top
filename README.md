## One-click install
It's on extensions.gnome.org :
https://extensions.gnome.org/extension/3004/apps-on-top/

## Manual install
To install, simply download as zip and unzip contents in ~/.local/share/gnome-shell/extensions/apps-on-top@RaphaelRochet
